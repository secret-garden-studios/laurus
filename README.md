# Laurus
Laurus is an animation framework for the web. The beta version is deployed [here](https://laurus.im) and is free to use without a credit card.

## Getting Started

The ability to run Laurus in your local development environment is intentionally limited.

No API keys are being distributed at the moment, and the backend repository is private, therefore no database interactions can be established. We do not have a standalone client that can run off mocked data.

This repository is made public for visibility, so anyone curious enough to be reading this file can poke around on GitHub or clone the code to their machine given some familiarity with client software that is built with Next.js, React, and TypeScript.
