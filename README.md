# Laurus
Laurus is an animation framework for the web. The beta version is deployed [here](https://laurus.im) and is free to use without a credit card.

## Getting Started

The ability to run Laurus in your local development environment is intentionally limited.

No API keys are being distributed at the moment, and the backend repository is private, therefore no database interactions can be established.

This repository is made public mainly for visibility. If you have some familiarity with Next.js/React/TypeScript tech stacks, then feel free to clone the code onto your machine and explore.
